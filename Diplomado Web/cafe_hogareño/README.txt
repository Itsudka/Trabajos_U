# ☕ Café Hogareño

Proyecto web desarrollado como parte del Diplomado en Desarrollo Web.  
Café Hogareño es un sitio web orientado a la presentación de un catálogo de cafés, reservas de mesas y contacto con clientes, enfocado en buenas prácticas de desarrollo frontend y backend básico.

---

## 🎯 Objetivo del proyecto

Desarrollar un sitio web responsive que implemente principios de diseño web moderno, uso de contenido multimedia, formularios funcionales y un backend básico, aplicando conceptos de metodologías ágiles y buenas prácticas.

---

## 🖥️ Interfaces del proyecto

El proyecto cuenta con las siguientes interfaces responsive:

- Página de inicio (Home)
- Catálogo de café
- Reservas de mesas
- Contáctanos

Todas las interfaces incluyen barra de navegación y footer.

---

## 🛠️ Tecnologías utilizadas

- **Git y GitHub**

---

## 🔁 Metodología de trabajo (Scrum y buenas prácticas)

Para el desarrollo del proyecto se aplicaron principios básicos de la metodología ágil **Scrum**, organizando el trabajo en iteraciones:

- **Sprint 1:** Maquetación inicial del sitio (HTML y Bootstrap)
- **Sprint 2:** Implementación del catálogo responsive y contenido multimedia
- **Sprint 3:** Desarrollo de formularios (contacto y reservas)
- **Sprint 4:** Implementación de backend básico en PHP y conexión con base de datos

Se aplicaron buenas prácticas como:
- Separación de responsabilidades (HTML, CSS, JS y PHP)
- Código legible y organizado
- Diseño responsive

## ▶️ Cómo ejecutar el proyecto

1. Clonar el repositorio:

2. Copiar el proyecto en la carpeta `www` de WAMP

3. Importar la base de datos desde phpMyAdmin

4. Iniciar los servicios de Apache y MySQL

5. Acceder desde el navegador
