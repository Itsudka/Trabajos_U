☕ Café Hogareño – Proyecto Web

Este proyecto es una página web estática creada como prototipo para una tienda de café y té, diseñada con HTML, CSS y Bootstrap 5.
Incluye un sistema de navegación, secciones informativas, catálogo visual de productos, formulario de contacto y un footer estilo profesional.

🚀 Tecnologías utilizadas

    HTML5

    CSS3

    Bootstrap 5.3

    Live Server (durante el desarrollo)

    Imágenes y estilos personalizados

🖥️ Puesta en marcha del proyecto (Live Server)

     utilizó la extensión Live Server de Visual Studio Code para levantar el proyecto en un entorno local.

🔧 Pasos para usar Live Server:

    Instalar la extensión Live Server desde el Marketplace de VS Code.

    Abrir la carpeta del proyecto en Visual Studio Code.

    Hacer clic derecho sobre index.html y seleccionar:
    "Open with Live Server"

    El sitio se abrirá automáticamente en:

    http://localhost:5500/


Live Server recargará la página automáticamente al guardar cambios (Ctrl + S).
